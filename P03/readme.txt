Вещественные числа записываются с точкой, а не с запятой

Программа запрашивает на ввод число переменных для инициализации (неотрицательное), затем по одной в строчке инициализирует вводимые переменные обязательно с начальными значениям.

Затем программа запрашивает выражение, которое необходимо вычислить (выражение до '\n'), затем вычисляет его, если не возникает ошибки.


Программа писалась и отлаживалась под Mac Os X Mojave 10.14 и под linux ubuntu
